# Weather App

A simple weather application built with **HTML, CSS, and JavaScript** as part of the
FreeCodeCamp JavaScript Algorithms and Data Structures curriculum.

This project fetches real-time weather data from a public API and displays current
conditions based on the selected city.

---

## Features

- Fetches weather data using an external API
- Displays temperature and weather conditions
- Basic dynamic UI updates
- Asynchronous data handling with `fetch`

---

## Notes

This project was developed as a learning exercise.
